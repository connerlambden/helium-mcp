# Contributing to Helium MCP Server

Thanks for your interest in contributing! This repo contains documentation, examples, and configuration for the Helium MCP Server. The server itself is hosted at [heliumtrades.com/mcp](https://heliumtrades.com/mcp-page/).

## How to Contribute

### Report a Bug

If you encounter an issue with the Helium MCP Server:

1. Check [existing issues](../../issues) to avoid duplicates
2. Open a [new issue](../../issues/new) with:
   - What you expected to happen
   - What actually happened
   - Your MCP client (Claude Desktop, Cursor, etc.)
   - Steps to reproduce

### Request a Feature

Have an idea for a new tool or improvement? Open a [feature request](../../issues/new).

### Improve Documentation

Found a typo, unclear instruction, or missing example? PRs are welcome:

1. Fork the repo
2. Create a branch (`git checkout -b fix/typo-in-readme`)
3. Make your changes
4. Submit a pull request
