# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the Helium MCP Server, please report it responsibly:

**Email:** [contact@heliumtrades.com](mailto:contact@heliumtrades.com)

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge your report within 48 hours and work to resolve the issue promptly.

## Scope

This policy covers:
- The Helium MCP endpoint (`https://heliumtrades.com/mcp`)
- This repository's documentation and configuration

## Disclosure

We follow responsible disclosure. Please do not publicly disclose vulnerabilities until we've had a chance to address them.
